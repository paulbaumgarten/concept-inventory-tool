CSV Format Reference


        
1. Test Specification CSV

        
The AI prompt above will generate this format. Headers must appear exactly as shown:

        question_number,topic_code,topic_name,question,answer_a,answer_b,answer_c,answer_d,correct_answer,misconception_a,misconception_b,misconception_c,misconception_d
        
    
          
  • question_number — sequential integer (1, 2, 3…)
    • 
          
  • topic_code — hierarchical code e.g. A1.1, B2.3
    • 
          
  • topic_name — human-readable section name (same for all questions with the same code)
    • 
          
  • question — full question stem text
    • 
          
  • answer_a through answer_d — the four answer choice texts (no letter prefix)
    • 
          
  • correct_answer — one of: A, B, C, or D
    • 
          
  • misconception_a through misconception_d — brief misconception description for each wrong answer; leave blank for the correct answer
    • 
        


        
2. Student Responses CSV (Google Forms export)

        
    
          
  • Column 1 — Timestamp (ignored)
    • 
          
  • Column 2 — Email address
    • 
          
  • Column 3 — Student name
    • 
          
  • Columns 4+ — One column per question, answers must be A, B, C, or D
    • 
        

        
The question columns must appear in the same order as in the test specification. Column headers are ignored for answer columns.

        
Google Forms tip: Set up your form so each answer option is just the letter A, B, C, or D (not the full answer text), or ensure the letter is the first character of each option.
